DevFest

Come join the largest community driven GDG event Seattle GDG DevFest at Google Fremont. There will be speakers, workshops, code labs and office hours on the topics of Firebase and Android. We will start out the DevFest with tech talks then transition into workshops, code labs and office hours. You can get help on how to migrate your existing Android apps to use Android Marshmallow features, or work start from scratch by following code labs.If you are new to Android, you can also get help on getting started with Android development.

At the end of the DevFest the best apps will present for a few minutes for show/tell. Don’t miss this great opportunity to learn from experts, and network with the other developers. Breakfast and lunch are included. You’ll have the chance to win some sweet swag in the process!

After the DevFest is complete and the swags have been awarded, it’ll be time to enjoy a few beers and network with fellow developers
